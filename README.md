# Location Search

### Built using React Native and Expo

## Install

* Step 1: https://docs.expo.io/versions/latest/introduction/installation/

* Step 2: `npm install`

* Step 3: `source env.sh`

* Step 4: `npm start`

* Step 5: Profit
